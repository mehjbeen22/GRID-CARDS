KNOW MORE ABOUT PROGRAMMING
The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
Cascading Style Sheet is a markup language responsible for how your web pages will look like. It controls the colors, fonts, and layouts of your website elements.
JavaScript is a scripting language that enables you to create dynamically updating content, control multimedia, animate images, and pretty much everything else
SASS (Syntactically Awesome Stylesheet) is a CSS pre-processor, which helps to reduce repetition with CSS and saves time. It is more stable and powerful CSS extension language that describes the style of document structurally.
jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website.
React is a JavaScript-based UI development library. Facebook and an open-source developer community run it.Although React is a library rather than a language, it is widely used in web development.
